<h1 align="center">Crowdplay</h1>


## Getting Started

### Important information:

Apache port: 8888

Mysql port: 8889

Database name: crowdplay

``` bash
# install dependencies
npm install

# run migration
php artisan migrate

# serve at localhost:8888
npm run dev

```
